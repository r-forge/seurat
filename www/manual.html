Introduction


data manager


At startup, the data manager window appears:

The data manager displays the different datasets which have been loaded. Detailed information about the files and the information they aer holding can be accessed by clicking on the names. In addition the manager window shows the objects described by the different datasets. These objects are e.g. genes, samples, CGH clones and chromosomes. Again, detailed information about the objects can be accessed by clicking on the names.

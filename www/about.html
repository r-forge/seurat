About Seurat



What is Seurat?


Seurat is Java based graphical user interface which provides interactive graphics together with explorative tools that allow for the integrative analysis of gene expression, array CGH (comparative genomic hybridization) and clinical data. To perform clustering and seriation algorithms seurat establishs a connection to the R statistical software via the Rserve client. Theoretically this connection allows the software to use all functions implemented in R and Bioconductor.

Features

  • Clustering Algorithms:
    • Hierarchical Clustering
    • kmeans Clustering

Download and Installation



R and Java Runtime Environment


In order to run Seurat the latest JRE (or JDK) and the latest R version need to be installed. In addition, to perform any clustering and seriation analysis, the R packages Rserve, amap and seriation need to be installed. This can be done by typing the following into the R console:

  install.packages(c("Rserve","amap","seriation"))

Installation and Run



To install just download the .zip file, unzip it and:



  • Windows: double click the Seurat.bat in the Win folder or type Seurat.bat in the command line.

  • Linux: launch the Seurat.jar file via the terminal by: 

    java -jar Seurat.jar


  • MacOS X: double click the Seurat.app in the MAC folder


Sample Data


Here are some sample data sets, which are ready to load and test with Seurat:

GeneExpression.txt    gene expression values of 7705 genes of 50 samples

ClinicalData.txt     16 clincial variables collected from the 50 samples

GeneAnnotations.txt     gene annotations of the 7705 genes

aCGHData.txt     CGH data for 7873 clones as measured by the 50 samples

The mapping of the 7873 CGH clones to the 7705 genes is time consuming, thus loading the array-CGH file will last some time (depending on your machine).

SampleData.zip

Most of the seriation algorithms (e.g. ARSA, BBURCG, BBWRCG, TSP, Chen) will not run with data sets of dimension 7000. To test the seriation algorithms, please try the short version of the data files:

GeneExpression.txt    gene expression values of 299 genes of 50 samples

ClinicalData.txt     16 clincial variables collected from the 50 samples

GeneAnnotations.txt     gene annotations of the 299 genes

aCGHData.txt     CGH data for 1144 clones (chromosome 1) as measured by the 50 samples
